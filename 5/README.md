Этот проект представляет собой простой js код для проверки на простое число

## Установка
1. Скопируйте репозиторий 
git clone https://github.com/malikbitirov/testt/5

## Использование
1. Перейдите в директорию проекта:
cd 5

2. Запуск проекта:
node index.js

node test.js
