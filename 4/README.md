## Функция суммироания
Этот проект представляет собой простой js код для подсчета суммы 

## Установка
1. Скопируйте репозиторий 
git clone https://github.com/malikbitirov/testt/

## Использование
1. Перейдите в директорию проекта:
cd testt/4

2. Запуск проекта:
node index.js

3. Запуск тестов проекта:
node test.js
